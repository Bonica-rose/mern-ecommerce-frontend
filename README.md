# Frontend of an Ecommerce Platform

A frontend part of an ecommerce application using the MERN Stack (MongoDB, Express.js, React.js and Node.js).

## Features
- User Authentication (Login & Register)
- Protected Routes
- Role-Based Access Control (Admin & User)
- Dashboard
- Necessay CRUD Operations
- Form Validation (React Hook Form + Yup)
- Product Search, Filter & Sorting
- Responsive UI (Tailwind CSS)
- Toast Notifications
- State Management (Redux Toolkit)
